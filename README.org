* Leader Key
The leader key is set to ~SPC~ (Space) in both normal and visual modes.

* File Operations
| Keybinding      | Command                  | Description                           |
|-----------------+--------------------------+---------------------------------------|
| ~SPC f h~       | ~my/consult-fd-home~     | Find files in home directory          |
| ~SPC f c~       | ~consult-fd~             | Find files in current directory       |

* Notes
- The leader key (~SPC~) works in both normal and visual modes
- ~jk~ is configured as an alternative to ~ESC~ for exiting insert mode (via evil-escape)
- In insert mode, you can use ~C-c~ followed by ~h/j/k/l~ for window navigation
- Many hydras allow repeated keypresses without re-entering the hydra
- Dired starts in insert state by default for easier file navigation
